Detailed Description

Custom column formatting information. Overrides the current culture regional information. Month names, days of week, and AM/PM are automatically localized. If undefined, the current culture regional information is used.

void CsvExcelExportImport.WorkbookProperties.RegionInfoOverride.SetAllValues< T > ( T  src)

If some or all the settings are already in a class, just bulk copy all the setting properties into this object by matching name and type. Mismatched properties are ignored. Any additional properties may be added separately or left to the default culture settings.
